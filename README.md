## Featured Project

### F3Commands (Fabric)
https://www.curseforge.com/minecraft/mc-mods/f3commands-fabric

**F3Commands** is a Fabric utility mod that exposes Minecraft debug-screen (F3) data through clean, intuitive slash commands.

**Key Features**
- Access biome, coordinates, FPS, light levels, and more via commands
- Removes reliance on the cluttered F3 debug screen
- Preserves immersion while keeping advanced data accessible
- Lightweight and optimized for modern Fabric loaders

Used by survival players, technical players, and mod developers.
